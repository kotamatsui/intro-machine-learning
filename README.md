# 機械学習概論（Introduction to Machine Learning）

| 講師  | 松井孝太 (医療統計学)|
| --- | -------- |
| 日程  | 水曜日3，4限|

## 概要
この講義では，機械学習の基礎理論と実践的な応用手法を学びます．教師あり学習・教師なし学習・深層学習の主要なアルゴリズムを理解し，Pythonを用いた実装を通じてデータ解析能力を身につけることを目的とします．講義の前半ではアルゴリズムの数理的基礎を扱い，後半では実データへの適用とグループ実習を行います．本講義の受講にあたり，各自のノートパソコン（Windows，Mac）を持参してください．

## 到達目標
- 機械学習の主要なアルゴリズム（回帰，分類，クラスタリング，次元削減，深層学習）の原理と特性を理解する．
- Pythonを用いて機械学習モデルの実装・学習・評価を適切に行い，結果を解釈できるようになる．
- 実データに対して適切な手法を選択し，解析結果を正確に報告・議論できるようになる．

## 成績評価
- 数回の個人レポートとグループによるプレゼンテーションと質疑応答を評価します．
- 配分はレポート70％，グループ実習でのプレゼンテーションと質疑応答30%相当とします．

## 準備
- [Python](https://www.python.org)（3.10以上）および[Jupyter Notebook](https://jupyter.org)（または[JupyterLab](https://jupyterlab.readthedocs.io)）を各自のパソコンにインストールします（インストールについては初回の講義時に説明します）．
  - Pythonは機械学習・データ科学の分野で最も広く使用されているプログラミング言語であり，豊富なライブラリエコシステム（NumPy，pandas，scikit-learn，PyTorchなど）が整備されています．
  - Jupyter Notebookはコード・テキスト・図を一つのドキュメントで管理できる対話型実行環境です．データ解析や実験の記録・共有に広く用いられています．
- 講義中のPythonコードの共有には[Google Colab](https://colab.research.google.com)を利用します．利用にはGoogleアカウントが必要となりますので，持っていない場合は作成をしてください（アカウントの作成は無料でできます）．
