## 概要
本講義では，生成AIの基盤技術である機械学習の基本的な諸概念（教師あり学習，教師なし学習，深層学習，生成モデル）を学ぶ．本講義を通して，生成AIの単なるユーザーから一歩踏み込んで，内部のメカニズムを意識しより深く理解できるようになることを目指す．また，これによって，生成AIの良い面だけでなく悪い面や限界を理解し，生成AIを含めた新規AI技術と適切に向き合えるようになることをめざす．

## 到達目標
-	「データを用いて未知の状況に汎化する」という機械学習の基本的な考え方，および機械学習を実行する際のフロー（訓練->検証->テスト）を理解する．
-	様々な機械学習モデルの特徴を理解する．
-	深層学習と生成AIのコンセプトと利用する上での注意点を理解する．
